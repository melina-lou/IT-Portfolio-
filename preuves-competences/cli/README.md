# Commandes réseau (CLI)

## Français

Pour montrer que je sais me servir de la ligne de commande Windows (ou PowerShell), j’ai capturé quelques commandes de base que tout technicien doit connaître.

### Commandes exécutées

**ipconfig /all** – Voir ma configuration IP, la passerelle, le serveur DNS.

**ping google.com** – Vérifier si j’ai accès à internet et mesurer la latence.

**nslookup google.com** – Voir l’adresse IP d’un nom de domaine et quel serveur DNS répond.

**tracert google.com** – Afficher le chemin des routeurs entre mon ordi et google.com (utile pour détecter où ça bloque).
