# Microsoft 365 – Administration de base

## Français

J’ai exploré l’environnement M365 (via un compte gratuit ou démo) et j’ai réalisé quelques actions courantes pour un technicien helpdesk.

### Ce que j’ai fait

1. **Création d’un utilisateur** dans le centre d’administration M365 (nom, prénom, nom d’utilisateur, rôle par défaut).
2. **Attribution d’une licence** (par exemple Office 365 E3) à cet utilisateur.
3. **Ajout d’un groupe** (ex. « Tous les utilisateurs de Montréal ») et ajout d’utilisateurs dedans.
4. **Partage d’un dossier OneDrive** avec un autre utilisateur, en donnant un accès « lecture seule ».
5. **Navigation dans SharePoint** pour voir la structure des sites d’équipe.

Chaque action a été capturée d’écran (dossier `captures/`).

### Compétences démontrées
- Administration de base M365
- Gestion des utilisateurs, groupes et licences
- Partage de fichiers OneDrive/SharePoint
