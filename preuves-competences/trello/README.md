# Trello – Gestion de tickets

## Français

Pour le Projet 1 (système de billetterie), j’ai utilisé Trello pour organiser mes tickets. Ce dossier contient une capture de mon tableau Trello.

### Ce que le tableau montre

- Colonnes : **À faire**, **En cours**, **Résolu**.
- Chaque ticket a un titre, une priorité (Haute, Moyenne, Basse), une description du problème, et des commentaires sur les actions prises.
- J’ai ajouté des étiquettes (couleurs) pour le type de problème : VPN, Mot de passe, Imprimante, Accès, etc.

### Capture
Le fichier `trello-board.png` dans ce dossier montre le tableau complet.

### Compétences
- Organisation de tâches
- Méthode Kanban
- Documentation de tickets
